# Termux-openai
Liberia de openai en python para uso en termux

<p>El uso del api de la libreria openai con la gestion de base de datos con sqlite3</p>

<h2>Instalacion</h2>

<p>git clone https://github.com/lalolanda11/Termux-openai</p>
<p>cd Termux-openai</p>
<p>chmod +x instalador.sh</p>
<p>./instalador.sh</p>
<p>python chat_boot.py</p>

<p>Para que puedas usar el api de termux con las mejoras actualizadas descarga el gestor de paquetes en fdroid</p>
<p>https://f-droid.org/F-Droid.apk</p>
<p>Instala la apk y Dale los permisos necesarios y luego actualiza la base de datos del gestor de paquetes entrando a la aplicaccion f-droid</p>
<p>Descarga la version mas reciente de termux y termux-api. Luego dale los permisos para necesarios a la api de termux</p>
